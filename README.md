# Study-Algorithm📔

## 스터디 규칙
* 격주로 3명이 백준 실버 단계 문제를 골라 백준 그룹에 추가. 
    * 첫주: 김나은, 배수연, 송수경
    * 다음주: 박규동, 김민기, 최은송
* 기한: 매주 금요일 자정까지 문제 올리기 
* 그 다음주 수요일까지(일 ~ 수) 백준 그룹에서 3문제 풀고 깃허브에 구현 방법 + 코드 짧게 올리기
* 금요일 전까지 각자 다른 스터디원이 푼 문제에 피드백 달기 


## 스터디 시간
* 매주 금요일 8시 ~ 11시
    * 2문제 골라 2시간동안 2문제 풀기
    * 남은 1시간 코드 리뷰 + 질문
    
    
## 폴더 구조 
* 자기 이름 폴더
   * 0월 0주차 폴더
     * 자바 소스 파일(사이트_난이도_번호.java)
