#백준 S1 1074 Z
○Idea
 - 2^N 크기의 2차원 배열에서 방문하는 순서를 출력하는 문제
 - 수업시간 풀었던 쿼드트리 문제와 유사해서 재귀 4번 호출해서 문제 해결
 - 종료조건 : 입력받은 r,c에 도달했을 때 값을 리턴
 - 인자로 넘겨주는 (row~row+n) (col~col+n) 범위에 r,c가 있을 경우(해당 사분면에 존재하는 경우) 재귀호출
 - 그 외에는 index값을 앞의 사분면만큼 증가 시킨다.

●평가
 - 처음에는 예전에 풀었던 문제가 생각나서 완탐으로 풀었으나 시간초과가 났다.
 - 시간 제한은 0.5초이고 2^15가 약 3만정도 되므로 완탐으로는 풀 수 없는 문제였다.
 - 타겟(r,c)에 해당하는 4분면만 선택하고 그 조건을 파악하는 것이 중요했다.

#백준 S3 14425 문자열 집합
○Idea
 - N개의 문자열이 정의되고 뒤에 M개의 문자열 중 포함되어 있는 문자열의 갯수를 구하는 문제
 - 원소들의 중복이 없고 순서는 상관없다 -> Set

●평가
 - Set 자료구조만 알고 있으면 바로 풀 수 있는 문제였다.
 - 다만 자바에서 set을 처음 사용해봐서 Set<T> = new HashSet<>()으로 객체를 생성한다는 것을 배웠다.

#백준 G3 1377 버블 소트
○Idea
 - 버블 소트로 주어진 코드에서 i를 출력하는 문제
 - i의 의미를 우선 파악 -> 버블 소트에서 flag가 false일 때 출력되는 i는 swap이 처음으로 일어나지 않는(정렬이 완료된)라운드를 의미한다.
 - 처음 배열과 정렬된 배열의 가장 큰 인덱스 차이(swap이 진행된 라운드 횟수)에 1을 더해준다.

●평가
 - 문제에서 i가 무엇을 의미하는지 파악하는 것이 가장 중요한 문제였다.
 - 버블 소트 자체가 가물가물해서 주어진 코드를 보고 풀었지만 소트의 종류와 동작 원리를 한번 정리 할 필요가 있다고 생각했다.
